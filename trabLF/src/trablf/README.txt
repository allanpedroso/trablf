 Para executar o programa execute a classe "AFD", após SIGA a sequência exibida no menu;

1. Para definir um alfabeto do autômato deve inserir os caracteres um a um.

2. As inclusões de estados são feitas automaticamente, só informar o número de estados que quer incluir;

3. As inclusões de transições são feitas uma a uma, deve informar estado origem, estado de destino e entrada aceita
   após isso volta ao menu principal, para adicionar outra transição só repetir o processo;

4. Para definir o estado inicial basta informar qual, obedecendo ao número que foi incluido anteriormente;

5. Para definir estados finais deve informar o estado, também um a um respeitando o número que foi incluido anteriormente;

6. Para executar o autômato insira a palavra respeitando o alfabeto que foi cadastrado;

7. Para minimizar o automato basta informar a opção, o programa removera todos os estados mortos.
